# أهداف تصميم تعبيرات فهم الموترات

كان الدافع الأصلي هو تقليص الموترات — تقليل أبعاد معينة بالجمع على متغيرات محددة.

## مثال: ضرب المصفوفات
$C_{i,k} = \sum_j A_{i,j} \cdot B_{j,k}$

```
[ i, k | { A[i, j] * B[j, k] } ]
```

## العلاقة مع اتفاقية جمع أينشتاين

| العملية | تعبير فهم الموتر |
|---------|-----------------|
| ضرب المصفوفات | `[ i, k \| { A[i,j] * B[j,k] } ]` |
| الأثر | `[ \| { A[i,i] } ]` |
| الضرب الخارجي | `[ i, j \| { a[i] * b[j] } ]` |
