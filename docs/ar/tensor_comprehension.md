# دليل تعبيرات فهم الموترات

## البنية
```rust
[ <فهارس> | <بنود> { <جسم> } ]
```

## 1. المولدات
```rust
let A = [i | i <- 0..5 { i }];     // صريح
let B = [i | { A[i] * 2 }];        // استدلال ضمني
```

## 2. الشروط (التصفية)
```rust
let Evens = [i | i <- 0..5, (i % 2) == 0 { i }];
```

## 3. الاختزال
المتغيرات المعرّفة في `<بنود>` والغائبة عن `<فهارس>` هي **متغيرات اختزال**. يكشف تحليل الاختزال الضمني (اتفاقية جمع أينشتاين) هذه المتغيرات تلقائيًا.

```rust
let dot = [i | i <- 0..1, k <- 0..N { A[k] * B[k] }];
```

## 4. أمثلة

### ضرب المصفوفات
```rust
let C = [i, j | i <- 0..M, j <- 0..P, k <- 0..N { A[i, k] * B[k, j] }];
```

## 5. الجسم الاختياري
```rust
let A = [i | i <- 0..5];                    // [0, 1, 2, 3, 4]
let grid = [i, j | i <- 0..N, j <- 0..M];  // موتر Shape(N, M, 2)
```
