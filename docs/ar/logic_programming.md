# البرمجة المنطقية في TensorLanguage (TL)

يدمج TL محرك استدلال منطقي قوي بأسلوب Prolog مباشرة في بيئة التشغيل الخاصة به. يتيح ذلك تعريف قواعد المعرفة وتنفيذ الاستدلال المنطقي ودمج النتائج مع الشبكات العصبية.

## 1. نظرة عامة على البنية

### الحقائق
```rust
father(alice, bob).       // "alice هو أب bob"
```

### إعلانات العلاقات
```rust
relation parent(entity, entity);
```

### القواعد
```rust
grandparent(x, z) :- father(x, y), father(y, z).
ancestor(x, y) :- father(x, y).
ancestor(x, y) :- father(x, z), ancestor(z, y).
```

### الاستعلامات
```rust
let is_father = ?father(alice, bob);
let children = ?father(alice, $child);
```

## 2. المخرجات الرمزية

يقوم TL تلقائيًا بربط أسماء الكيانات بمعرفات عددية فريدة. عند العرض، يتم تحويل المعرفات تلقائيًا إلى الأسماء الأصلية.

## 3. النطاق وتنظيم الملفات

يجب تعريف الحقائق والقواعد في **النطاق العام**. يسمح نظام الوحدات (`mod`، `use`) بالتنظيم في ملفات متعددة.

## 4. التكامل مع الموترات

نتائج الاستعلام هي موترات TL قياسية يمكن استخدامها مباشرة في العمليات الرياضية وعمليات الشبكات العصبية.

## 5. النفي والمقارنات الحسابية

### النفي
```rust
manager(X) :- employee(X), not(has_boss(X)).
```

يدعم TL النفي الطبقي. يتم اكتشاف الدورات السلبية وتسبب خطأ في التجميع.

### المقارنات الحسابية
```rust
adult(X) :- person(X, Age), Age >= 18.
```

المعاملات المدعومة: `>`، `<`، `>=`، `<=`، `==`، `!=`
