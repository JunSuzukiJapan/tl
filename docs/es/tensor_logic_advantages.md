# Tensor Logic: Por qué puede resolver más problemas que los tensores solos

## Resumen

**Tensor Logic** integra el cálculo tensorial con el razonamiento lógico, añadiendo **estructura lógica**, **satisfacción de restricciones** y **razonamiento simbólico**.

## Limitaciones de los tensores tradicionales
- Optimización discreta difícil
- Restricciones lógicas no expresables naturalmente
- Razonamiento simbólico no soportado

## Nuevas clases de problemas
- **Optimización combinatoria**: N-Queens, TSP, coloración de grafos, SAT
- **Problemas inversos**: Juego de la vida inverso, renderizado inverso
- **Razonamiento estructurado**: Completación de grafos de conocimiento, síntesis de programas

## Ventajas

| Aspecto | Tensores tradicionales | Tensor Logic |
|---------|----------------------|--------------|
| Representación | Puramente numérica | Lógica + numérica |
| Restricciones | Términos de regularización | Soporte nativo |
| Problemas discretos | Difícil de aproximar | Relajación continua |
| Razonamiento simbólico | No soportado | Convertido a operaciones tensoriales |
