# 支持的模型训练机制

TensorLanguage (TL) 除了强大的张量运算之外，还支持神经网络模型训练。本文档解释了从定义模型到实现训练循环以及使用 TL 保存训练模型的工作流程。

## 1. 基本培训概念

TL 训练按以下步骤进行：

1. **模型定义**：使用`struct`来定义保存参数和状态的层和模型。
2. **前向传递**：计算输入张量的输出以获得预测分数或逻辑。
3. **损失计算和向后传递**：在损失函数（例如，`cross_entropy`）上调用 `loss.backward()` 来计算每个参数的梯度。
4. **优化**：对每个参数调用内置优化函数（例如 `adam_step`）来更新它们，并使用 `Tensor::clear_grads()` 重置梯度。

## 2. Tensor 和 GradTensor 的区别

TL 有两种用于数值计算和训练的主要静态张量类型：

- **`Tensor<T, R>`**：标准多维数组数据。它不跟踪梯度（计算历史），因此速度快且内存效率高。它主要用于**推理期间的数据处理**和存储**优化器内部状态（例如动量和方差）**。
- **`GradTensor<T, R>`**：用于训练的梯度跟踪张量。它记录计算过程（构建计算图），并在调用“backward()”时执行自动微分来计算梯度。您必须始终使用“GradTensor”来让优化算法学习/更新**参数（权重、偏差等）。
