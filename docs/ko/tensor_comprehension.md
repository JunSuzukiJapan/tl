# 텐서 내포 표현 가이드

TL 언어는 Haskell의 리스트 내포 표현에서 영감을 받은 강력하고 표현력 있는 텐서 내포 표현 구문을 제공합니다.

## 구문 개요

```rust
[ <indices> | <clauses> { <body> } ]
```

*   **`<indices>`**: 출력 텐서의 차원을 나타내는 변수 목록.
*   **`<clauses>`**: 생성기(범위 지정) 또는 조건식(필터) 목록.
*   **`{ <body> }`**: 각 요소 값을 계산하는 블록 식.

---

## 1. 생성기 (Generators)

```rust
let A = [i | i <- 0..5 { i }];     // 명시적
let B = [i | { A[i] * 2 }];        // 암시적 추론
```

## 2. 조건식 (Filtering)

```rust
let Evens = [i | i <- 0..5, (i % 2) == 0 { i }];
// 결과: [0, 0, 2, 0, 4]
```

## 3. 리덕션 (Reductions)

`<clauses>`에서 정의되었지만 출력 `<indices>`에 **포함되지 않는** 변수는 **리덕션 변수**로 처리됩니다. 이는 **암시적 리덕션 분석(아인슈타인 합 규약)**으로 구현되어 본체 식을 분석하여 리덕션 변수를 자동 감지합니다.

```rust
let dot = [i | i <- 0..1, k <- 0..N { A[k] * B[k] }];
```

## 4. 추가 예시

### 행렬 곱셈
```rust
let C = [i, j | i <- 0..M, j <- 0..P, k <- 0..N { A[i, k] * B[k, j] }];
```

### 대각 요소 추출
```rust
let Diag = [i | i <- 0..N, j <- 0..N, i == j { Matrix[i, j] }];
```

## 5. 선택적 본체 (Optional Body)

```rust
let A = [i | i <- 0..5];                    // [0, 1, 2, 3, 4]
let grid = [i, j | i <- 0..N, j <- 0..M];  // Shape(N, M, 2) 텐서
```
