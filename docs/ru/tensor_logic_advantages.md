# Tensor Logic: Почему он может решать больше задач, чем тензоры в одиночку

## Обзор

**Tensor Logic** интегрирует тензорные вычисления с логическим выводом, добавляя **логическую структуру**, **удовлетворение ограничений** и **символьные рассуждения**.

## Ограничения традиционных тензоров
- Дискретная оптимизация затруднена
- Логические ограничения не выражаются естественно
- Символьные рассуждения не поддерживаются

## Новые классы задач
- **Комбинаторная оптимизация**: N-Queens, TSP, раскраска графов, SAT
- **Обратные задачи**: Обратная «Игра жизни», обратный рендеринг
- **Структурированные рассуждения**: Дополнение графов знаний, синтез программ

## Преимущества

| Аспект | Традиционные тензоры | Tensor Logic |
|--------|---------------------|--------------|
| Представление | Чисто числовое | Логика + числа |
| Ограничения | Регуляризационные члены | Нативная поддержка |
| Дискретные задачи | Трудно аппроксимировать | Непрерывная релаксация |
| Символьные рассуждения | Не поддерживаются | Преобразуются в тензорные операции |
