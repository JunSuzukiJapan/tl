# Руководство по тензорным включениям

## Синтаксис
```rust
[ <индексы> | <клаузулы> { <тело> } ]
```

## 1. Генераторы
```rust
let A = [i | i <- 0..5 { i }];     // Явный
let B = [i | { A[i] * 2 }];        // Неявный вывод
```

## 2. Условия (Фильтрация)
```rust
let Evens = [i | i <- 0..5, (i % 2) == 0 { i }];
```

## 3. Редукции
Переменные из `<клаузул>`, отсутствующие в `<индексах>`, являются **переменными редукции**. Неявный анализ редукции (соглашение Эйнштейна о суммировании) автоматически обнаруживает эти переменные.

```rust
let dot = [i | i <- 0..1, k <- 0..N { A[k] * B[k] }];
```

## 4. Примеры

### Умножение матриц
```rust
let C = [i, j | i <- 0..M, j <- 0..P, k <- 0..N { A[i, k] * B[k, j] }];
```

## 5. Необязательное тело
```rust
let A = [i | i <- 0..5];                    // [0, 1, 2, 3, 4]
let grid = [i, j | i <- 0..N, j <- 0..M];  // Тензор Shape(N, M, 2)
```
